# WRDS Cloud

WRDS Cloud is a HPC service where it is possible to process the data avaliable on [WRDS- Wharton Research Data Services]().

WRDS Cloud is only available for CBS staff.

More information can be found [here](https://www.cbs.dk/en/library/databases/wrds-wharton-research-data-services).
